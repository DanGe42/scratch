About this site

This site was built to visualize the progress of all floors in Kings Court English College House in the annual NSO house-wide scavenger hunt.

Data source

The data originates from a spreadsheet to which only KCECH staff have access. That said, these are not the official rankings. If the data seems to be outdated, please ask your RA/GA to update the spreadsheet.

Technology stack

This site runs on a Python backend and utilizes Highcharts to render the chart. This is currently hosted from Amazon EC2.

Oh, by the way, fork me on Github!
